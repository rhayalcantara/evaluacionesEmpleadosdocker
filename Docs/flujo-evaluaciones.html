ADMIN
Crear y activar período
▼

📅Crear el período

  • Ruta: Configuración → Periodos → Agregar
  • Definir descripción, fecha inicio y fecha fin
  • Elegir Tipo: "Evaluación de Fin de Año (Puntuada)"
  • Marcar activa = true para activarlo
/Periodo

⚖️Configurar pesos PDC

  • Ruta: Configuración → Porciento Desempeño/Competencia
  • Agregar Desempeño 30% para este período
  • Agregar Competencia 70% para este período
  • Sin PDC configurado, totalCalculo será 0
/porciento-desempeno-competencia
2
ADMIN
Agregar competencias al período
▼

🎯Configurar competencias en el período activo

  • Ruta: Configuración → Períodos de Evaluación
  • El sistema carga el período activo automáticamente via GET /api/Periods/activo
  • Aquí se vinculan los Goals / Objetivos al período
  • Se pueden filtrar por puesto (PositionSecuential)
/evaluation-periods

📋Crear / editar competencias individuales

  • Ruta: Configuración → Competencias
  • Cada competencia pertenece a un Grupo de Competencias
  • Se asocia a puestos específicos o a todos
/Objetivo
3
ADMIN
Cargar objetivos (KPIs / KRIs / Proyectos)
▼

📤Carga masiva desde archivo

  • Ruta: Configuración → Carga de Archivos KPIs
  • Subir archivo Excel/CSV con indicadores de desempeño
  • Endpoint: POST /api/EmpleadoDesempenoes/upload
  • Asocia los objetivos a cada empleado por período
/uploadfile

📈Carga manual de KPIs y KRIs

  • Gestionar KPIs individualmente: Configuración → KPIs
  • Gestionar KRIs individualmente: Configuración → KRIs
  • Gestionar Proyectos/Objetivos: Plan Estratégico → Proyectos/Objetivos
/kpi · /kri · /opp

🤝Asignar metas a subordinados

  • El supervisor asigna metas específicas a cada miembro
  • Se accede desde el Home o desde Evaluar Equipo
  • Formulario: SupervisorGoalsComponent
/supervisor-goals

Completar autoevaluación AutoEvaluado
▼

🧭Acceso al formulario

  • Ruta: Evaluaciones → AutoEvaluación
  • El sistema detecta el período activo desde localStorage
  • Si no existe evaluación, se crea en estado Borrador
  • Componente: FormEvaluationEmployeComponent

📝Metas de desempeño

  • Registrar el % de logro de cada KPI / KRI / Objetivo
  • Indicar medio de verificación y comentario
  • El sistema calcula resultadoLogro = logro × peso / 100

⭐Competencias (escala numérica 1–5)

  • Calificar cada competencia con valor del 1 al 5
  • El componente criterialitem calcula el promedio
  • Genera puntuacioncompetenciacolaborador
💾
Grabar → estado: AutoEvaluadoSe calculan desempeño y competencia del colaborador y se persisten.

SUPERVISOR
Evaluar al colaborador EvaluadoPorSupervisor
▼

👥Acceder al equipo

  • Ruta: Evaluaciones → Evaluar Equipo
  • Ve la lista de subordinados directos con su estado
  • Hace clic en la tarjeta del colaborador a evaluar
/EvaluarSubordinados

⭐Calificar competencias del supervisor

  • Calificar cada competencia del 1 al 5 (perspectiva supervisor)
  • Ver y validar las metas registradas por el colaborador

Grabar → estado: EvaluadoPorSupervisorSe calcula el score del supervisor. El totalCalculo pondera 20% colab + 80% supervisor.

SUPERVISOR
Enviar evaluación al colaborador Enviado
▼

📨Botón "Enviar" en la tarjeta del colaborador

  • Aparece solo cuando el estado es EvaluadoPorSupervisor
  • Componente: CardEmpleadoComponent → método Enviar()
  • Cambia el estado a Enviado y notifica el cambio
  • El colaborador verá la sección de "Aceptación de Evaluación"
/EvaluarSubordinados → botón Enviar
Fase de cierre
7
COLABORADOR
Revisar y aceptar la evaluación Completado
▼

👀Ver la evaluación del supervisor

  • El colaborador ingresa a AutoEvaluación — ve el estado Enviado
  • Aparece la sección "Aceptación de Evaluación"
  • Confirma si hubo entrevista con el supervisor
/Evaluacion → sección aceptación

✍️Opciones de aceptación

  • Acepta en Conformidad — está de acuerdo con la evaluación
  • Acepta en No Conformidad — no está de acuerdo (requiere escribir comentario)
  • Método: onAceptarEvaluacion() en FormEvaluationEmployeComponent
🏆
Aceptar Evaluación → estado: Completado Se genera automáticamente el PDF de la evaluación y se redirige al Home.

Configuración del Administrador

Pasos que el admin debe completar antes de iniciar cualquier tipo de evaluación

📅

1. Crear Período

Define el período, sus fechas y elige el tipo: Final de Año o Medio Año. Márcalo activo para que los empleados lo vean al ingresar a AutoEvaluación.

/Periodo
⚖️

2. Configurar Pesos PDC

Asigna los porcentajes de Desempeño y Competencia para el período. Sin esto, el totalCalculo será 0.

/porciento-desempeno-competencia
🎯

3. Agregar Competencias

Configura las competencias del período en Períodos de Evaluación. Vincula Goals a puestos. Aparecerán en los formularios del empleado y supervisor.

/evaluation-periods · /Objetivo
📤

4. Cargar Objetivos

Sube los KPIs, KRIs y Proyectos vía archivo masivo o carga individual. Estos son los indicadores cuantitativos que el empleado reportará su avance.

/uploadfile · /kpi · /kri · /opp
🤝

5. Asignar Metas

El supervisor asigna las metas específicas a cada miembro de su equipo usando el panel de Supervisor Goals desde el Home.

/supervisor-goals
📚

6. Catálogo de Cursos

Para la evaluación de medio año, el colaborador sugiere hasta 3 cursos. Carga el catálogo de cursos disponibles con anticipación.

/cursos-capacitacion
📊

7. Monitorear Estado

Usa los reportes para ver quién completó, quién está pendiente y el resumen de resultados. Filtra por período para ver el avance en tiempo real.

/ReporteEstadoEvaluacion · /Reportes
⚠️

Importante: Período Activo

Solo el período con activa=true aparece en AutoEvaluación. El formulario (Final vs Medio Año) se detecta automáticamente por periodo.tipo.

💡 Reglas del sistema

Un período activo a la vez. Si se activa otro, los empleados verán el nuevo al recargar.

La evaluación se crea automáticamente la primera vez que el empleado accede a AutoEvaluación para un período.

Solo el supervisor puede Enviar. El empleado no puede completar hasta recibir el envío del supervisor.

No Conformidad solo en Final de Año. En medio año el cierre es directo: "Completar Evaluación".

📊 Cálculo — Final de Año

// Desempeño individual por meta
logro_i = logro% × peso_i / 100
promedioDesempeno = Σ logro_i

// Scores por actor
colabScore = desempeno×PDC% + compColab×PDC%
superScore = desempeno×PDC% + compSuper×PDC%

totalCalculo = colabScore×.20 + superScore×.80

💬 Cálculo — Medio Año

// Solo avance de objetivos (cuantitativo)
avance_i = logro% × peso_i / 100
desempenoFinal = Σ avance_i × PDC%

// Competencias = texto (sin score)
competencias = observaciones narrativas

totalCalculo = desempenoFinal
// (puntuación de competencias = 0)
