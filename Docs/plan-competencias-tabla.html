Tabla de Evaluación de Competencias — Medio Año

Reemplazar tarjetas por tabla con dropdowns Colaborador / Supervisor

¿Por qué este cambio?

La sección "Evaluación de Competencias" en form-evaluation-medio-ano muestra actualmente una tarjeta por competencia con textareas de observación, pero no tiene dropdown de calificación, aunque los campos repuesta y repuestasupervisor ya existen en el modelo IGoalEmpleadoRespuesta y son procesados por el controller. El formulario de referencia (Excel + imagen) requiere una tabla simple: nombre de competencia · calificación colaborador · calificación supervisor.

// Opciones de calificación para tabla de competencias
public readonly calificaciones: { valor: number; etiqueta: string }[] = [
  { valor: 0, etiqueta: '— Seleccione —' },
  { valor: 1, etiqueta: '(1) Deficiente' },
  { valor: 2, etiqueta: '(2) Necesita Mejorar' },
  { valor: 3, etiqueta: '(3) Cumple' },
  { valor: 4, etiqueta: '(4) Muy Efectivo' },
  { valor: 5, etiqueta: '(5) Excelente' },
];
Por qué [ngValue]: repuesta es number. Con [value] Angular convierte a string y el binding pierde el tipo. [ngValue] preserva el número exacto.

Eliminar el bloque completo de tarjetas y sustituir por:

<!-- 3. EVALUACIÓN DE COMPETENCIAS -->
<div class="my-4 text-dark"
     *ngIf="evaluacionempleado?.evaluacionGoals?.length > 0">
  <h2>Evaluación de Competencias</h2>
  <div class="table-responsive">
    <table class="table table-bordered competencias-table">
      <thead><tr>
        <th class="col-competencia">Competencia</th>
        <th class="col-calificacion text-center">Colaborador</th>
        <th class="col-calificacion text-center"
            *ngIf="supervisor || evaluacionempleado.estadoevaluacion === 'Completado'">
          Supervisor
        </th>
      </tr></thead>
      <tbody>
        <tr *ngFor="let met of evaluacionempleado.evaluacionGoals; let index = index"
            [class.fila-alt]="index % 2 !== 0">
          <td class="align-middle">{{ met.goal.objetivo.nombre }}</td>
          <td class="text-center align-middle">
            <select class="form-select form-select-sm competencias-select"
                    [(ngModel)]="evaluacionempleado.goalEmpleadoRespuestas[index].repuesta"
                    name="comp-rating-colab-{{index}}"
                    [disabled]="supervisor || sololectura">
              <option *ngFor="let op of calificaciones" [ngValue]="op.valor">{{ op.etiqueta }}</option>
            </select>
          </td>
          <td class="text-center align-middle"
              *ngIf="supervisor || evaluacionempleado.estadoevaluacion === 'Completado'">
            <select class="form-select form-select-sm competencias-select"
                    [(ngModel)]="evaluacionempleado.goalEmpleadoRespuestas[index].repuestasupervisor"
                    name="comp-rating-super-{{index}}"
                    [disabled]="!supervisor || sololectura">
              <option *ngFor="let op of calificaciones" [ngValue]="op.valor">{{ op.etiqueta }}</option>
            </select>
          </td>
        </tr>
      </tbody>
    </table>
  </div>
</div>
La columna Supervisor usa la misma condición *ngIf que el resto del formulario (líneas 42, 130, 274…). El [disabled]="!supervisor || sololectura" garantiza que un colaborador viendo una evaluación completada no pueda editar la columna del supervisor.
